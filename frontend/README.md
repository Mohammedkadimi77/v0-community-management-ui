# Community Management UI

This is a React project powered by [Vite](https://vite.dev).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

You can start editing the pages in the `app/` directory and the router in `src/App.tsx`.
